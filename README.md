# princewitherdick.github.io

GitHub user site for [PrinceWitherdick](https://github.com/PrinceWitherdick), served at
https://princewitherdick.github.io/.

Its main purpose beyond a simple landing page is to publish **host-root** AI-training and
text-and-data-mining opt-out signals, which are only honored when served from a host root:

- [`robots.txt`](robots.txt) - blocks known AI-training crawlers across the whole host.
- [`ai.txt`](ai.txt) - AI-training opt-out declaration.
- [`.well-known/tdmrep.json`](.well-known/tdmrep.json) - TDM Reservation Protocol opt-out.
- `<meta name="robots" content="noai, noimageai">` in [`index.html`](index.html).

Full policy: [AI Training and Data-Mining Notice](https://github.com/PrinceWitherdick/stonetop/blob/main/AI-TRAINING-NOTICE.md).
